YouTube Mindmap

YouTube Mindmap is an innovative tool that transforms YouTube content into structured knowledge. By generating concise summaries and detailed mindmaps, it enables users to uncover actionable insights from individual videos or link multiple videos on a consistent subject or speaker into a cohesive strategy. The integrated semantic search ensures users find the best content for their specific needs, while the multilingual support broadens accessibility. Originally developed as part of the Empowerbot project, YouTube Mindmap extends the chatbot’s capabilities, offering a powerful resource for learning, strategizing, and problem-solving.

×

АI Agent for Public Grants

This project developed a Grant Application AI Agent that streamlines the process of matching grant applications with expert reviewers. The system takes a grant application, generates a tailored job description for an ideal reviewer, and identifies the most suitable expert based on a similarity index using Vertex AI. By incorporating human oversight in the review process, the system ensures higher accuracy and fairness in grant approvals. The back-end is built with Django and Python, while the front-end is powered by React, creating a seamless interface for users to engage with the system.

×

FlexiText AI

FlexiText is an AI-based tool designed to simplify document modification. Users can upload files in Word or PDF format and use Vertex AI to perform tasks such as summarizing, translating, or making textual modifications. After processing, the user can download the edited file in the original format (Word or PDF). The platform is designed for ease of use with a clean, responsive front-end built with HTML and CSS, while the back-end is powered by Flask and Python to efficiently handle file uploads, AI interactions, and file generation.

La Clé Intranet Site

The La Clé Intranet Site is a fully functional internal platform built for a French company, providing employees with essential tools for productivity and communication. Key features include user authentication with secure login/logout functionality, a time management system for tracking work hours, and a robust notification and announcement module to keep staff informed about company updates. The system is built on Django with a clean user interface, providing an efficient and user-friendly experience for managing internal operations.
